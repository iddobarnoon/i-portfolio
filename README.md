## ğŸš¦ Getting Started

### Prerequisites
- Node.js (version 14.0 or higher)
- npm or yarn

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/[YOUR_USERNAME]/Portfolio.git
   ```

2. Navigate to the project directory
   ```bash
   cd Portfolio
   ```

3. Install dependencies
   ```bash
   npm install
   # or
   yarn install
   ```

4. Start the development server
   ```bash
   npm run dev
   # or
   yarn dev
   ```
